# 🐍 Snake Game

A classic Snake game built with vanilla HTML, CSS and JavaScript — featuring 5 progressive stages, themed obstacles, and a pixelated apple.

🎮 **[Play the game here](https://felixlie27.github.io/snake-game)**

---

## 🕹️ How To Play

- Press any **arrow key** to start the game
- Use **arrow keys** to control the snake
- Eat the 🍎 **apple** to grow and score points
- Each apple = **10 points**
- Reach **100 points** to advance to the next stage
- **Don't hit** the walls, obstacles, or yourself!

---

## 🏆 Stages

| Stage | Theme | Obstacles | Speed |
|-------|-------|-----------|-------|
| 1 | 🌿 Beginner | 3 rocks | Slow |
| 2 | 🌲 Forest | 6 tree stumps | Medium |
| 3 | 🌋 Volcano | 10 lava rocks | Fast |
| 4 | 🌊 Deep Sea | 14 corals | Very Fast |
| 5 | 👑 Final Boss | 18 crystals | Extreme |

---

## 🎯 Scoring

| Action | Points |
|--------|--------|
| Eat apple | +10 pts |
| Complete a stage | Carry score forward |
| Complete all 5 stages | 🏆 500 pts max |

---

## 💀 Game Over

- Hit a **wall** → Game Over
- Hit an **obstacle** → Game Over  
- Hit **yourself** → Game Over

Your total score carries across all stages. Beat all 5 to win!
